# Steps to run

1. first clone the repo
2. change dir to repo
3. then create venv via python python3 -m venv venv
4. then activate the venv (in linux: source venve/bin/activate)
5. then install the dep via pip install -r requirements.txt
6. after then config the pdf raw file path with the path of raw pdf file and raw_pdf_exctracted_files_path with the path that you want to save the extracted iamges
7. by default this script create a directory with the name of pdf file and save each pdf file to its directory
8. run the script by : python3 pdf2image.py
